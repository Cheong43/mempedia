# ReAct与传统方法对比

## Summary

ReAct相较于纯Chain-of-Thought和纯Action方法的核心优势与差异

## Details

纯Chain-of-Thought仅进行内部推理，缺乏外部信息源，无法获取实时数据，容易产生知识截止问题导致的错误。纯Action仅执行操作但缺乏推理规划，容易在复杂信息中迷失。ReAct结合两者优势，通过显式推理指导行动选择，通过行动结果反馈修正推理，形成闭环决策机制。
