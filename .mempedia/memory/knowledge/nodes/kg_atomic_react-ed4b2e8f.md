# React (前端框架)

## Summary

Meta(Facebook)开发的用于构建UI的JavaScript库

## Details

核心特性包括：1)组件化架构（可复用的独立UI单元）；2)虚拟DOM（Diff算法提升渲染性能）；3)JSX语法（JS中嵌入类HTML标记）；4)单向数据流（可预测的状态管理）。由Meta维护，需配合Babel、Webpack等工具链使用，是当前主流前端技术栈核心。
