# ReAct (Reasoning + Acting)

## Summary

推理与行动交替进行的智能体模式

## Details

核心思想是通过Thought（思考）和Action（行动）的循环来解决任务。与CoT不同，ReAct强调与外部环境交互，可以调用工具获取信息。适用于需要外部工具调用、实时数据获取的场景。
