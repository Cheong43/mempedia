# Self-Refine

## Summary

自我精炼，迭代的自我反馈优化循环

## Details

流程：生成初稿→自我批评（找出问题）→根据反馈改进→重复直到满意。适用于文本润色、代码优化、创意写作。与Reflexion类似但更注重迭代优化而非错误学习。
