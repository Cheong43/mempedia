# Role-Play

## Summary

角色扮演，Agent扮演特定角色提供专业回答

## Details

Agent扮演特定角色（资深工程师、医生、代码审查员、苏格拉底导师）以获得更专业回答。可与Multi-Agent结合实现多角色互动（如律师vs法官）。
