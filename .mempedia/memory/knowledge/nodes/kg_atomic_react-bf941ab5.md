# ReAct框架

## Summary

结合推理(Reasoning)与行动(Acting)的AI Agent框架，由Google Research于2022年提出

## Details

核心思想是让AI通过Thought(思考)→Action(行动)→Observation(观察)的循环迭代解决复杂任务。与纯Chain-of-Thought(仅推理)和纯Action(仅行动)相比，ReAct具备可追溯性、准确性、鲁棒性和交互性优势。典型应用场景包括实时信息查询、多步推理任务和工具调用场景。
