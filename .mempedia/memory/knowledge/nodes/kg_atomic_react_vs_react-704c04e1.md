# React vs ReAct 歧义

## Summary

技术术语中'React'可能指前端库或AI推理框架

## Details

区分特征：ReAct(AI框架)强调Reasoning+Acting循环，用于LLM决策；React(前端库)强调Component+Virtual DOM，用于UI渲染。当用户询问'react框架'而无上下文时，应同时介绍两者并询问具体领域（前端开发vs AI应用）。
