# Multi-Agent

## Summary

多智能体系统，多角色协作完成任务

## Details

多个Agent分别扮演不同角色（研究者、分析师、写作者、审核者、协调者），通过顺序协作（流水线）、并行协作（汇总）或讨论协商（达成共识）完成任务。适用于复杂项目管理、内容创作、研究分析。每个Agent内部可嵌套ReAct模式。
